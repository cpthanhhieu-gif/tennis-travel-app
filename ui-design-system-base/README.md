# UI Design System — Base Template
> Bộ chuẩn dùng cho mọi dự án. Source tham khảo: Mantine Design System (mantinedev/mantine)
> Mỗi dự án tạo 1 bản copy riêng, fill brand tokens vào cột "Brand Value"

## Cấu trúc
```
ui-design-system-base/
├── tokens/
│   ├── colors.md
│   ├── typography.md
│   ├── spacing.md
│   ├── effects.md
│   └── breakpoints.md
├── components/
│   ├── button.md
│   ├── card.md
│   ├── form.md
│   ├── typography.md
│   ├── navigation.md
│   └── feedback.md
└── rules/
    ├── accessibility.md
    ├── layout-rules.md
    ├── interaction-rules.md
    └── naming-conventions.md
```

## Cách dùng
1. Copy toàn bộ folder này vào dự án mới
2. Đổi tên thành `ui-design-system/`
3. Fill cột "Brand Value" trong từng file token
4. Xóa cột "Default Value" sau khi đã fill xong
