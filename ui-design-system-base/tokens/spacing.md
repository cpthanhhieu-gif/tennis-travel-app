## Semantic Spacing (Brand)
| Token | Giá trị | Tailwind | Dùng cho |
|---|---|---|---|
| page-margin-mobile | 16px | `px-4` | Margin 2 bên trên mobile |
| page-margin-desktop | 24px | `px-6` | Margin 2 bên trên desktop |
| section-gap | 48px | `gap-12` hoặc `mb-12` | Khoảng cách giữa các section |
| component-gap | 24px | `gap-6` | Khoảng cách giữa components |
| element-gap | 12px | `gap-3` | Khoảng cách giữa elements trong component |
| inline-gap | 8px | `gap-2` | Khoảng cách icon-text |
| card-padding | 24px | `p-6` | Padding nội card |
| bottom-nav-height | 64px | `h-16` | Bottom navigation bar |
| top-nav-height | 56px | `h-14` | Top header |
