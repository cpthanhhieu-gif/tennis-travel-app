# UI Design System — App Travel

## Cấu trúc

```
ui-design-system/
├── tokens/              # Giá trị nền tảng (màu, font, spacing...)
│   ├── colors.md
│   ├── typography.md
│   ├── spacing.md
│   ├── effects.md
│   └── breakpoints.md
│
├── components/          # Quy tắc từng component
│   ├── button.md
│   ├── card.md
│   ├── form.md
│   ├── typography.md
│   ├── navigation.md
│   └── feedback.md
│
└── rules/               # Nguyên tắc tổng quát
    ├── accessibility.md
    ├── layout-rules.md
    ├── interaction-rules.md
    └── naming-conventions.md
```

## Cách dùng
- Điền thông tin vào từng file trước khi apply vào code
- Mọi thay đổi UI phải tham chiếu về đây trước
- File này là nguồn sự thật duy nhất (single source of truth)
